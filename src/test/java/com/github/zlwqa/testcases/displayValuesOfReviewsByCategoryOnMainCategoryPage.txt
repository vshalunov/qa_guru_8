        Title: Отображение категорий "Смартфоны", "Камерофоны", "Флагманы", "Планшеты", "Смарт-часы",
        "Гаджеты", "Android" в обзоре по категориям на странице "Смартфоны и гаджеты"

        Severity: Minor
        Priority: Medium
        Tags: Web

        Preconditions: Открыта страница https://www.mvideo.ru/

        Steps to reproduce:
        1. Нажать на кнопку "Каталог"
        2. В боком меню нажать на категорию "Смартфоны и гаджеты"
        3. Пролистать страницу до заголовка "Обзоры по категориям"

        Expected Result:
        В обзорах по категориям отображаются категории
        "Смартфоны", "Камерофоны", "Флагманы", "Планшеты", "Смарт-часы", "Гаджеты", "Android"
